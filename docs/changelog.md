## [0.1.0] - YYYY-MM-DD

<!-- TODO: Add release date when 0.1.0 is released -->

### Added

- Initial release of SPIMquant
- Deformable registration to templates (ABAv3, gubra, MBMv3, turone, MouseIn)
- Atlas-based quantification of pathology
- Support for multiple segmentation methods (threshold, otsu+k3i2)
- Group-level statistical analysis
- Cloud storage support (S3, GCS)
- Coiled integration for cloud execution
- High-resolution Imaris crop creation
- BIDS App compliance
- Snakemake-based workflow

### Features

- OME-Zarr input support
- Multi-stain processing
- Parallel processing with Snakemake and Dask
- Quality control outputs
- HTML report generation
- Configurable templates and parameters
